# Shared utility folder

This folder contains the following files that contain constants, types, and utility functions that are used throughout the rest of the repo:

- `event_driven.py`:  Shared utilities for the event driven simulations

- `graph_creation.py`: Utility functions for creating graphs

- `monitorying.py`: Utilities for recording runtime, memory consumption, and energy consumption of a Python code block

- `simulation_config.py`: Dataclasses for describing the parameters to be used during a simulation
